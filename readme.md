# Lista de Tarefas

Projeto de **Lista de Tarefas** desenvolvido com **HTML, CSS e JavaScript**, com foco em organização de código, manipulação do DOM e persistência de dados utilizando **Local Storage**.

---

## Funcionalidades

- Adicionar novas tarefas
- Definir prioridade da tarefa:
- Urgente
- Importante
- Normal
- Excluir tarefas da lista
- Persistência de dados com Local Storage
- Interface simples e organizada
- Código JavaScript dividido em módulos

---

## Conceitos Utilizados

- HTML semântico
- CSS Flexbox
- JavaScript ES6+
- Modules (`import` / `export`)
- Manipulação do DOM
- Local Storage
- Eventos (`addEventListener`)
- Organização de pastas e arquivos

## Local Storage

As tarefas são armazenadas no **Local Storage**, garantindo que:

- Os dados não sejam perdidos ao recarregar a página
- A lista de tarefas permaneça salva entre sessões
